v0.3.0 — What's New

Bigger than a feature update.

Version 0.3.0 adds the full management layer — history, users, MCP, audit, notifications, and more. Here are the highlights.

New

🧙 Setup Wizard

5-step web wizard for first-time configuration. Set your API key, password, channels, and active backend — no YAML editing, no SSH.

New

💬 Conversation History

SQLite-backed history viewer with full-text search. Filter by backend, channel, user, or date range. Delete individual conversations.

New

👥 User Management

JWT + RBAC with three roles: admin, operator, viewer. Create, update, and delete users from the dashboard. No code changes needed.

New

🔌 MCP Server Manager

Add, test, and remove MCP servers via UI. Test connectivity before committing. Works alongside LibreChat-compatible MCP endpoints.

New

📈 Cost & Token Chart

7-day rolling Canvas chart with per-model breakdown. Metrics polled every 15 seconds, stored in SQLite, visualized in the dashboard.

New

🔍 Local AI Scanner

One click detects Ollama, LM Studio, llama.cpp, Jan AI, and vLLM on your network. Adds them to your routing table automatically.

New

🎯 Routing Rules Builder

Condition-based routing: send channel A to Opus 4.8, channel B to a local Llama model. Priority-ordered rules, toggle per-rule.

New

🎭 Preset Personas

Ship with Customer Support, Code Reviewer, and Personal Assistant templates. Clone, edit system prompt, and assign to any channel.

New

🛒 ClawHub Skill Browser

Browse and one-click install from 5,700+ community skills. Search by keyword or category without leaving the dashboard.

New

🔔 Notification Center

Bell icon with unread badge. Backend health alerts, update notifications, and webhook alerts surface here — not buried in logs.

New

📋 Audit Log

Every user action — login, config change, backend restart — is written to SQLite with IP, timestamp, and result.

New

💾 Backup & Restore

Download a .tar.gz archive of all data (SQLite DB + config) from the settings tab. Restore by extracting into /data — no CLI needed.

📱 Mobile-responsive — 640px breakpoint, 48px touch targets on all interactive elements
⚡ Sticky progress bar — live feedback for model pulls, migrations, and long operations
🤖 Headless installer — pass SC_API_KEY, SC_PASSWORD, etc. as env vars for zero-interaction CI deploys
📡 Channel management UI — configure all 50+ channels from the browser, no YAML editing required
