# One-liner — Ubuntu, Debian, any Linux

# Or Docker Compose
$ git clone https://github.com/hbkdad/selfclawy && cd selfclawy
$ cp .env.example .env && nano .env && docker compose up -d

Why SelfClawy

Everything Clawy offers.
On your own server.

useclawy.com charges monthly to host OpenClaw. SelfClawy gives you the same setup on hardware you control — free, forever.

⚡

One-command deploy

Single curl command auto-installs Docker, configures channels, generates secrets, and starts the gateway. Under 2 minutes.

🔒

Your data, your server

Memory files, conversation history, and credentials stay on your machine. Nothing phones home. No vendor lock-in.

📊

Management dashboard

Dark-mode web UI with live log streaming, start/stop/restart controls, and health checks — no SSH needed after setup.

🔄

Auto-updates

Enable Watchtower to keep OpenClaw current automatically, or run docker compose pull manually.

🤖

Any AI model

Claude, GPT-4o, Gemini, Mistral, local Llama via Ollama. Swap models by changing one env var.

🛠️

Fully hackable

MIT licensed. Extend with OpenClaw skills, write your own, or fork the whole thing. No permission needed.
