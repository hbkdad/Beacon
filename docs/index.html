Your AI that works

Self-host OpenClaw on any VPS. Message it from WhatsApp, Telegram, Discord. It clears your inbox, manages your calendar, runs commands — 24/7. No SaaS fees. You own everything.

Deploy in 2 minutes → ⭐ GitHub 
# One-liner install
$ curl -fsSL https://raw.githubusercontent.com/hbkdad/selfclawy/main/scripts/install.sh | bash

# Or Docker Compose
$ git clone https://github.com/hbkdad/selfclawy && cd selfclawy
$ cp .env.example .env && nano .env && docker compose up -d

# Your lobster is live at http://YOUR_IP:18789 🦞
Why SelfClawy

Everything Clawy offers.
On your own server.

useclawy.com charges monthly to host OpenClaw. SelfClawy gives you the same setup on hardware you control — free.

⚡

One-command deploy

Single curl sets up Docker, pulls OpenClaw, configures channels, starts the gateway. Under 2 minutes.

🔒

Your data, your server

Memory files, conversation history, credentials stay on your VPS. Nothing leaves your machine.

📊

Management dashboard

Dark-mode web UI to start/stop/restart, stream live logs, check health — no CLI needed.

🔄

Auto-updates

Optional Watchtower keeps OpenClaw current, or run docker compose pull manually.

🤖

Any AI model

Claude, GPT-4o, Gemini, Mistral, Llama via Ollama — swap in .env, no code changes.

🛠️

Fully hackable

MIT licensed. Extend with OpenClaw skills from ClawHub, or write your own.
