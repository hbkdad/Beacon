Why Beacon

Everything you need.
Nothing you don't.

Beacon is the only open-source dashboard that unifies OpenClaw (302K stars), Hermes Agent, and Ollama in one dark-mode control plane — with every management feature built in.

🧙

Setup Wizard

5-step web wizard for first-time configuration. Set API keys, channels, and active backend with zero SSH after deploy.

📡

Live Log Streaming

Real-time backend logs streamed via Socket.io. Tail OpenClaw, Hermes, or Ollama without touching the command line.

💬

Conversation History

SQLite-backed history viewer with full-text search. Filter by backend, channel, user, or date range. Delete individual conversations.

🛒

ClawHub Skills (5,700+)

Browse and one-click install from 5,700+ community skills. Search by keyword or category without leaving the dashboard.

🔌

MCP Server Management

Add, test, and remove MCP servers via UI. Test connectivity before committing. No config file editing required.

📊

Cost & Token Tracking

7-day rolling chart with per-model breakdown. Metrics polled every 15s, stored in SQLite, visualized in the dashboard.

🎯

Model Routing Rules

Condition-based routing: send channel A to Opus, channel B to a local Llama model. Priority-ordered rules, toggle per-rule.

🎭

Preset Personas

Ship with Customer Support, Code Reviewer, and Personal Assistant templates. Clone, edit, and assign to any channel.

👥

User Management

JWT + RBAC with three roles: admin, operator, viewer. Create, update, and delete users entirely from the dashboard UI.

📋

Audit Log

Every action — login, config change, backend restart — written to SQLite with IP, timestamp, and result. Full accountability.

📈

Prometheus Metrics

Expose a /metrics endpoint in Prometheus text format. Plug directly into Grafana or any compatible scraper.

🔍

Local AI Scanner

One click detects Ollama, LM Studio, llama.cpp, Jan AI, and vLLM on your network. Adds them to your routing table automatically.

Get Started

One command. Full stack.

Copy and run the installer on any Linux server with Docker. A browser-based wizard handles everything from there — no YAML, no SSH, no guesswork.
